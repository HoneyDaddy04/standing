# Standing

An AI-native compliance and legal firm for Nigerian companies. CAC filings, FIRS
and VAT returns, board resolutions and everyday contracts, kept in good standing
for a flat fee. AI does the heavy lifting, a licensed professional signs off.

This is a product demo: a marketing landing page plus a functional dashboard app
shell (compliance calendar, documents, a secure chat with live PII redaction, and
a simulated drafting pipeline).

## Live

https://honeyDaddy04.github.io/standing/

## Develop

```bash
npm install
npm run dev      # http://localhost:5185
npm run build    # production build to dist/
```

Built with Vite, React and TypeScript. Deployed to GitHub Pages via the workflow
in `.github/workflows/deploy.yml` on every push to `main`.
